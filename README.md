# Software Project Showcase

This repository provides an overview of some of the software projects I have worked on during my Software Engineering degree at Aalborg University.

Many of these projects were developed in collaboration with other students. Some repositories are private or owned by other group members, so this repository serves as a portfolio describing my contributions.

# Food Planner

### Technologies
Flutter • C# • REST API • Figma • Git • SCRUM

A mobile application developed for children with autism to support meal planning through an accessible and user-friendly interface.

### My contributions

- Frontend development in Flutter
- Implemented reusable UI components
- Developed meal creation features
- Implemented a customized camera feature
- REST API integration
- Participated in code reviews
- Contributed to wireframing and UI discussions
- Worked closely with stakeholders throughout development

# Symmetry-Aware Matformer (Bachelor Project)

### Technologies

### My contributions

- Developed parts of the data pipeline
- Implemented symmetry-aware crystal transformations
- Structured and validated scientific datasets
- Automated testing using PyTest
- Debugging and performance evaluation
- Collaborated on model design and implementation
